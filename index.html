Project One: Closet Search Filter

  • Client Request: As I was choosing what sorts of projects I wanted to include in my portfolio, I decided to look within my own personal dilemmas. For the past couple months I have been living as a bit of a nomad which meant I was essentially living out of my suitcase. I found myself rummaging through my clothes to decide on an outfit, to then reorganize, to then rummage again the next day. To avoid a mess I would sometimes rewear a lot of the items that were sitting on top, forgetting all about the rest of my precious wardrobe.
  • Solution: I decided to use JavaScript React to create a closet search filter. This allowed me to be able to scroll through my items as well as filter the results depending on type or color. I included functions such as "pin" (which requires simply clicking the item) and "unpin" in order to see different clothing items next to each other. There is also a hide button to prevent the item from appearing wheather it's because it's a no or because it's in the dirty bin.
  • Skills Performed: HTML/CSS, JavaScript React, filtering arrays, Class Based Components, use of constructor/super functions to set state

Project Two: Personal Online Blog

  • Client Request: For my next project, I wanted to create another website that I could use everyday. Something I make sure to do often is journal - whether it's recalling a happy event or reflecting on current emotions, I have found it to be very beneficial for my own being. I do like to journal by hand, but sometimes if I'm in a time crunch or have lots to say I turn to typing. I will usually just use my notes App however wanted to create something that had more of a journal feel, something seperate from where I keep my daily to do lists.
  • Solution: I decided to create a simple online blog platform using JavaScript React as well as Google Firebase. For the homepage I decided on a photo of a journal along with some writing I did on SketchPad to create that handwritten journal feel. Under "thoughts" I have the entries being transferred from Firebase appearing in order from newest to oldest. Lastly, you can create a new entry under the "new" tab that will add onto the Firebase data collection at publish. I kept the design clean and simple embodying that of an empty notebook.
  • Skills Performed: HTML/CSS, JavaScript React, React Hooks: useState and useEffect, Google FireBase
